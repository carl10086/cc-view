# cc-view

Claude Code 数据可视化仪表盘。

## 项目结构

- `app/` — Next.js App Router 页面
- `lib/` — 工具函数和数据层
- `components/` — React 组件
- `types/` — TypeScript 类型定义
- `refer/` — 参考项目目录（软链接，gitignore）

## refer 目录说明

`refer/` 是三方参考项目的软链接，用于源码参考和学习。当前指向 `claude-code-haha`。

## 技术栈

- Next.js 16 + React 19
- TypeScript
- Tailwind CSS v4
- shadcn/ui
